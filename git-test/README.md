# git日志查看
 ```git log``` 查看提交历史
 ```git log --oneline``` 简单的显示提交历史
 ```git log --graph``` 开启了拓扑图，显示分支合并等信息
 ```git log --reverse --oneline``` 逆向显示所有日志
 ```git log --author=提交的作者 --oneline -5``` 查找某个用户提交的日志信息及个数
 [更多日志操作信息](https://git-scm.com/docs/git-log)
